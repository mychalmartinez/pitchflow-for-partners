# Pitchflow for Partners

Repeatable agency pitch page system for Partner Account Manager workflows. Each pitch is a single-page, CMS-driven, brand-themed Webflow site targeting C-suite and key decision-makers at mid-market to enterprise brands.

├── canonical-reference.md         # Source of truth: page structure, CMS schema, skill architecture

## MAST Design System

The `mast/` folder contains the branded design system foundation for this site, generated by the `mast-brand-injector` skill.

**What's already done:**
- 130 variables created across 5 collections (Color, Layout, Typography, Theme, Components)
- Components and Styles XSCP pages ready to paste into Webflow Designer

**To paste the MAST pages into Webflow:**
1. Serve the preview locally: `cd mast/preview && python3 -m http.server 8787`
2. Open `http://localhost:8787`
3. Click **Copy Components XSCP** → open Webflow Designer on the Components page → `Cmd+V`
4. Click **Copy Styles XSCP** → open Webflow Designer on the Styles page → `Cmd+V`
5. Manually add a `Poppins` FontFamily variable to the Typography collection (headings)

## Skills

Install skills by unzipping `.skill` and `.zip` files into `~/.claude/skills/`. All four are pre-installed if you've run setup.

| Skill | Invoke | Purpose |
|-------|--------|---------|
| `mast-brand-injector` | `/mast-brand-inject` | Inject brand tokens → Webflow variables + XSCP pages |
| `webflow-core` | (auto) | Core Webflow building workflows and MCP reference |
| `page-cloner` | `/page-cloner` | Scrape + decompose any site page into sections |
| `webflow-migration-collection-builder` | (auto) | Build CMS collection schemas from migration plans |
